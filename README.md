# Wontu Frontend

Angular frontend for Wontu.

## Requirements

- Node.js: 20.19+ (22 LTS recommended)
- pnpm: 9+

## Setup

1. Install pnpm if needed:

```bash
npm install -g pnpm
```

2. Install dependencies:

```bash
pnpm install
```

## Running the Dev Server

1. Start the app:

```bash
pnpm start
```

2. Open: http://localhost:4200
